# ⌚ Luxury Watches Website

A modern and responsive **Luxury Watches Website** developed using **HTML5** and **CSS3** as part of my **Full Stack Development** learning journey.

## 🚀 Project Overview

This project showcases a premium watch collection with a clean and elegant user interface. It focuses on building responsive layouts, attractive sections, and a smooth browsing experience using only HTML and CSS.

## ✨ Features

* Responsive Hero Section
* Watch Features Section
* Customer Review Section
* Bestselling Watches Showcase
* Modern Card Layout
* Add to Cart Buttons
* Elegant Footer
* Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
